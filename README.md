# True Click SEO — Static Website

Premium digital agency website for True Click SEO.  
Built with HTML, CSS, and JavaScript. PHP contact form handler included.

## Tech Stack
- HTML5 (semantic, SEO-first)
- Vanilla CSS (custom design system)
- Vanilla JavaScript (no frameworks)
- PHP (contact form backend)

## Pages
- Homepage (`index.html`)
- About (`about.html`)
- Services (`services.html`)
- Portfolio (`portfolio.html`)
- Case Studies (`case-studies.html`, `case-study.html`)
- SEO Service (`seo.html`)
- Website Design (`website-design.html`)
- Website Development (`website-development.html`)
- Technical SEO (`technical-seo.html`)
- Insights / Blog (`insights.html`)
- Contact (`contact.html`)

## Deployment
Upload all files to `public_html` on Hostinger.
PHP contact handler: `contact-handler.php`
Contact email: `trueclickseo@gmail.com`

## Local Preview
Open `index.html` directly in any browser.
